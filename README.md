# Koseki's Skills

This repository is a collection of "Skills" that contains system prompts and workflow instructions tailored for AI development agents.

Built upon several excellent open-source projects, some of the knowledge and instructions from these references are used as-is, while others have been optimized and modified to fit my (Koseki's) personal development style and conform to `CLAUDE.md` conventions.

## References

The following projects were referenced and inspired the creation of this repository:

- [forrestchang/andrej-karpathy-skills](https://github.com/forrestchang/andrej-karpathy-skills)
- [mattpocock/skills](https://github.com/mattpocock/skills)
- [obra/superpowers](https://github.com/obra/superpowers)
- [awslabs/aidlc-workflows](https://github.com/awslabs/aidlc-workflows)
